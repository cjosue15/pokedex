# Pokedex

## ✍🏻 Description

Pequeño proyecto realizado con react y components styled, consumiendo la pokeapi.

## 🚀 Get started

Clone project, download dependencies with

```
npm install
```

run on your local

```
npm start
```

## 🎨 Demo

You can see the demo here → [Pokedex](https://pokedexreactapp.netlify.app/)
